# 🏥 Healthcare Patient Data Analysis using SQL

## 📌 Project Overview

This project focuses on analyzing a healthcare patient dataset using **MySQL**. The objective is to extract meaningful business insights related to patient demographics, hospital operations, billing, insurance providers, medical conditions, and doctor performance using SQL queries.

The project demonstrates core SQL concepts that are commonly used by Data Analysts in real-world healthcare analytics.

---

## 🎯 Objectives

- Analyze patient demographics
- Explore hospital admission trends
- Evaluate billing and revenue performance
- Compare insurance providers
- Analyze doctor performance
- Identify common medical conditions
- Generate healthcare business insights using SQL

---
